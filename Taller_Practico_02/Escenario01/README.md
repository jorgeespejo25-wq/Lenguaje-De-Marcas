# Escenario
*"Eco-Logistics" (Gestión de Flotas)*
**El reto.** Necesitamos controlar una flota de reparto de última milla.
## Consultas de Negocio.
1. Listado visual de vehículos con batería crítica (< 15%) y estado "En ruta" para enviar aviso de retorno.

2. Informe de paquetes cuyo ID no cumple el patrón de control (detección de errores de escaneo).

3. Panel de control con el sumario de vehículos agrupados por su estado actual.
